Understanding AI Agents

A short, hands-on course grounded in your EnM (Energy & Materials) scenarios. Work through it in order; each lesson is one tangible win. Ask your teacher (chat) anything along the way.

6 lessons Hands-on Grounded in EnM scenarios

At the heart of every lesson is one idea: an agent runs a loop. It perceives the situation, reasons about what to do, acts by calling a tool, then observes the result — and loops again until the job is done.

Perceiveread the input / current state
Reasonthe LLM decides what to do next
Act (tool)call a function / run code
Observeread the tool result… then loop
