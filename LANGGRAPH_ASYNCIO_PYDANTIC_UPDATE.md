# LangGraph AsyncIO and Pydantic Interview Labs

Added two dedicated interview-focused learning pages:

- `genai-portal/langgraph-asyncio.html`
- `genai-portal/langgraph-pydantic.html`

Integration updates:

- Linked both labs from `genai-portal/modules/12_langgraph.html`.
- Linked both labs from the LangGraph component reference.
- Added both pages to unified navigation and search.
- Expanded LangGraph search keywords.

The AsyncIO lab covers event-loop fundamentals, structured concurrency, `TaskGroup`, cancellation, timeouts, semaphores, blocking-code diagnosis, async nodes, `ainvoke`, `astream`, interview Q&A, quizzes, and exercises.

The Pydantic lab covers Pydantic v2 models, strict configuration, validation and serialization, field/model validators, discriminated unions, tool contracts, LangGraph state choices and current limitations, boundary validation, interview Q&A, quizzes, and exercises.
