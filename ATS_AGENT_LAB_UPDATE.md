# ATS Agent Build Lab update

Added a new seven-lesson applied course under `genai-portal/ats-agent-lab/`. The material was derived from the uploaded ATS hackathon source, including its six agent modules, shared Claude wrapper, Pydantic contracts, service orchestration, tests, runtime configuration and architecture documents.

The course deliberately distinguishes bounded single-task LLM workers from open-ended autonomous agents, and covers implementation, technology choices, deterministic safeguards, privacy, human control, optimization, evaluation and a guided FastAPI capstone.
